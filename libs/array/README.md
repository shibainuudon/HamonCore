# Hamon.Array

このライブラリはC++標準ライブラリの`<array>`を拡張・改良する機能を提供します。

## `<array>`からの変更点

* 可能な限り`constexpr`をつけた
* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした
* 関数を追加

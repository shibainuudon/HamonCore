# Hamon.Functional

このライブラリはC++標準ライブラリの`<functional>`を拡張・改良する機能を提供します。

## `<functional>`からの変更点

* 可能な限り`constexpr`をつけた
* C++の特定のバージョン以降で追加された機能をバージョンに関係なく使えるようにした
* いくつかの関数・クラスを追加

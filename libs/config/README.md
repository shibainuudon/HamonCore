# Hamon.Config

このライブラリはプラットフォーム、コンパイラ、C++バージョンなどの差を吸収するための機能を提供します。

### 定義されるマクロの例

| マクロ             | 説明               | 値の例                 |
| ------------------ | ------------------ | ---------------------- |
| HAMON_PLATFORM     | プラットフォーム名 | "Win32",               |
| HAMON_ARCHITECTURE | アーキテクチャ名   | "x86_64"               |
| HAMON_COMPILER     | コンパイラー名     | "Microsoft Visual C++" |
| HAMON_CXX_STANDARD | C++バージョン      | 20                     |

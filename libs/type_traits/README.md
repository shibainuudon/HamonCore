# Hamon.TypeTraits

このライブラリは型の特性を判定・操作するためのクラスを定義します。
C++標準ライブラリの`<type_traits>`を拡張・改良します。

## `<type_traits>`からの変更点

* C++の特定のバージョン以降で追加されたクラスをバージョンに関係なく使えるようにした
* クラスを追加

# Hamon.TypeTraits

このライブラリはC++標準ライブラリの`<type_traits>`を拡張・改良する機能を提供します。

## `<type_traits>`からの変更点

* C++の特定のバージョン以降で追加されたクラスをバージョンに関係なく使えるようにした
* いくつかのクラスを追加

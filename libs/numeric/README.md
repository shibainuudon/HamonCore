# Hamon.Numeric

このライブラリはC++標準ライブラリの`<numeric>`を拡張・改良する機能を提供します。

## `<numeric>`からの変更点

* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした

# Hamon.Algorithm

このライブラリはC++標準ライブラリの`<algorithm>`を拡張・改良する機能を提供します。

## `<algorithm>`からの変更点

* 可能な限り`constexpr`をつけた
* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした
* 関数を追加

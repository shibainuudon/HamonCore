# Hamon.Compare
このライブラリはC++標準ライブラリの`<compare>`を拡張・改良する機能を提供します。

## `<compare>`からの変更点

* 可能な限り`constexpr`をつけた
* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした

# Hamon.Bit
このライブラリはC++標準ライブラリの`<bit>`を拡張・改良する機能を提供します。

## `<bit>`からの変更点

* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした
* 可能な限り`nodiscard`をつけた
* いくつかの関数を追加

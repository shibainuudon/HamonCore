# Hamon.Optional

このライブラリはC\+\+標準ライブラリに`C++17`で追加された`<numbers>`を、C\+\+11でも使えるようにするものです。

## `<optional>`からの変更点

* 可能な限り`constexpr`をつけた
* 可能な限り`nodiscard`をつけた

# Hamon.Iterator
このライブラリはC++標準ライブラリの`<iterator>`を拡張・改良する機能を提供します。

## `<iterator>`からの変更点

* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした

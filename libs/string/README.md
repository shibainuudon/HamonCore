# Hamon.String

このライブラリはC++標準ライブラリの`<string>`を拡張・改良する機能を提供します。

## `<string>`からの変更点

* 可能な限り`constexpr`をつけた
* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした

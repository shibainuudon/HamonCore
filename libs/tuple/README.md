# Hamon.Tuple

このライブラリはC++標準ライブラリの`<tuple>`を拡張・改良する機能を提供します。

## `<tuple>`からの変更点

* C++の特定のバージョン以降で追加された関数をバージョンに関係なく使えるようにした
* いくつかの関数を追加
